Core4 Concept Technical Definition Real-Life Analogy Scratch Example Python Example
Variables Name given to hold an unknown or known quantity known as a value A variable is defined by setting something equal to something else. It can be thought of as a bucket that can be filled with different amounts of water
Loops Sequence of instructions that is continually repeated until a certain conition is reached. An exmaple could be a while loop, or if else statement. This can be seen in real life in doing homework. You do one assignment and then check to see if there is another assignment left to do. If there is then you complete it. If not, then you can go to sleep.
Conditionals This is when if something happens, then it prompts another action. This can be seen by if statements or booleans. In real life, if you're hungry, then you will eat.
Functions A function defines a certain set of actions. A function is defined by using 'def ----' In real life, everyone could be picking up buckets of water, but the sizes of their buckets may be different.
